Neovim Cheatsheet

The essential commands for beginners and beyond.

The Three Modes

NORMAL Mode

For navigation and manipulation. Press Esc to enter.

INSERT Mode

For typing text. Enter with i, a, o, etc.

VISUAL Mode

For selecting text. Enter with v, V, or Ctrl+v.

Move cursor to other end

Run operators like d, y, c on selected text.

The Vim Grammar: Operator + Motion/Text Object

Combine an operator with a motion or text object. For example: ciw = Change Inner Word.

Operators (Verbs)

d - delete

y - yank (copy)

c - change (delete & insert)

> / < - indent / un-indent

Text Objects (Nouns)

iw / aw - inner / a word

i" / a" - inner / around quotes

i) / a) - inner / around parentheses

i} / a} - inner / around braces

ip / ap - inner / a paragraph

Examples

d$ - Delete to end of line

caw - Change a word (with space)

yi" - Yank text inside quotes

Vap - Select a paragraph visually

>i} - Indent inside curly braces
