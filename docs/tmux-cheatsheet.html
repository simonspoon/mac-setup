Tmux Cheat Sheet

Your quick guide to terminal multiplexing mastery.

Default Prefix: Ctrl + b

Press this combination before most commands.

Sessions

Start a new session

tmux

Start a new, named session

tmux new -s [name]

List active sessions

tmux ls

Attach to the last session

tmux a

Attach to a named session

tmux a -t [name]

Kill a session

tmux kill-session -t [name]

Detach from current session

Ctrl + b d

Rename current session

Ctrl + b $

Windows

Create new window

Ctrl + b c

Next window

Ctrl + b n

Previous window

Ctrl + b p

Go to window by number

Ctrl + b [0-9]

Rename current window

Ctrl + b ,

List windows (choose from list)

Ctrl + b w

Kill current window

Ctrl + b &

Panes

Split pane horizontally

Ctrl + b "

Split pane vertically

Ctrl + b %

Switch between panes

Ctrl + b ← ↑ → ↓

Close current pane

Ctrl + b x

Toggle pane zoom

Ctrl + b z

Show pane numbers

Ctrl + b q

Resize pane (hold Ctrl)

Ctrl + b Ctrl+↑↓←→

Swap with previous/next pane

Ctrl + b { or }

Cycle through pane layouts

Ctrl + b Space

Break pane into a new window

Ctrl + b !

Copy Mode

Enter copy mode

Ctrl + b [

Quit copy mode

q

Start selection

Space

Copy selection

Enter

Paste from buffer

Ctrl + b ]

List paste buffers

Ctrl + b =

Miscellaneous & Other

Show key bindings (help)

Ctrl + b ?

Enter command prompt

Ctrl + b :

Show clock in current pane

Ctrl + b t

Reload config file

tmux source-file ~/.tmux.conf

Created with love for the terminal. Happy multiplexing!
